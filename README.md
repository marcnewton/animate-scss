# animate-scss
Animate SCSS

https://daneden.github.io/animate.css/

Converted Animate.css to scss for inclusion in scss projects.

## VUE 2 JS

```vue
<transition name="custom-transition" enter-active-class="animated fadeIn" leave-active-class="animated fadeOut">
	<div v-if="displayed">
		Hello World
	</div>
</transition>
```
